# 🔐 Simple Password Generator

Привет! Это мой учебный проект на **Python**. Простой и надежный генератор безопасных паролей.

## ✨ Особенности
* 🐍 Написано на чистом Python.
* 🔡 Использует буквы, цифры и спецсимволы.
* 🛠️ Можно выбрать любую длину пароля.
* 🇬🇧 Имена переменных в коде на английском (лучшие практики!).
